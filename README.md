具身智能飞行员视觉仪表识别系统 — 技术演示

Embodied Pilot Vision Instrument Recognition System — Technical Demo

## 简介

具身智能飞行员视觉仪表识别系统硬件项目的前端技术演示。涵盖低延迟视觉采集、仪表识别、多区域感知、机器人部署及复杂环境鲁棒性评估。

## 功能

- **全景座舱视角** — 动态蓝天白云 + 风挡框架，随飞行姿态实时变化
- **12 种飞行仪表** — 空速表、高度表、姿态仪、HSI、升降速度表、转弯侧滑仪、发动机N1、EGT、燃油量、滑油压力、襟翼指示、起落架
- **头部追踪控制** — 接入摄像头，用面部位置实时控制飞机俯仰/横滚/航向
- **4 种飞行场景** — 白昼巡航 / 夜间飞行 / 湍流环境 / 低能见度
- **检测框标注** — 模拟仪表识别框和置信度显示
- **多区域感知** — 6 个座舱区域同步感知状态
- **机器人部署** — 3 台机器人在线状态及延迟
- **环境鲁棒性** — 6 种环境条件下识别精度评估
- **全链路延迟分析** — 五阶段延迟分解 + 实时趋势图

## 使用方式

直接打开 `index.html`，或用任意 HTTP 服务器托管：

```bash
python -m http.server 3000
# 访问 http://localhost:3000
```

### 操作

| 操作 | 说明 |
|---|---|
| 点击"头部追踪" | 开启摄像头控制飞机姿态 |
| 头部左右移动 | 飞机横滚（侧身） |
| 头部上下移动 | 飞机俯仰（抬头/低头） |
| 空格键 | 重新校准头部追踪 |
| 按键 1-4 | 切换飞行场景 |
| 点击标签页 | 切换飞行场景 |

## 技术栈

- 纯 HTML/CSS/JS，零外部依赖
- Canvas 2D 渲染
- FaceDetector API（Chrome 原生）
- WebRTC getUserMedia 摄像头接入
- 响应式布局（1920px → 移动端）
