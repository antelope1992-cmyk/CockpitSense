 具身智能飞行员视觉仪表识别系统 — 技术演示

Embodied Pilot Vision Instrument Recognition System — Technical Demo

## 简介

具身智能飞行员视觉仪表识别系统（v4.0）的硬件项目前端技术演示。模拟飞行员在座舱中的完整交互体验，涵盖低延迟视觉采集、仪表识别、飞行动力学仿真、头部追踪控制、飞行训练任务、注意力分析、多区域感知、机器人部署及复杂环境鲁棒性评估。

## 核心目的

- **技术验证**：展示具身智能系统在航空座舱场景中的视觉仪表识别能力
- **人机交互**：通过头部追踪实现自然的飞行姿态控制，模拟飞行员操作范式
- **训练评估**：提供多种飞行训练任务，实时评估飞行员表现和仪表注意力分配
- **环境适应性**：验证系统在多种光照、气象和振动条件下的鲁棒性

## 功能

### 座舱视觉仿真
- **全景座舱视角** — 动态天空渐变、星空、云层、地形、跑道，随飞行姿态实时变化
- **风挡框架结构** — 完整的座舱风挡拱形框架 + 中央支柱 + 侧窗支柱 + 遮光板
- **12 种飞行仪表** — 空速表、高度表、姿态仪、水平状态仪(HSI)、升降速度表、转弯侧滑仪、发动机N1、EGT、燃油量、滑油压力、襟翼指示器、起落架指示器

### 飞行动力学模型
- **物理仿真** — 基于 C172 简化模型，包含推力、阻力、重力分量的能量交换模型
- **协调转弯** — 根据坡度角自动计算转弯率
- **发动机参数联动** — 油门影响 N1 转速、EGT 温度、滑油压力和燃油消耗
- **湍流扰动** — 湍流场景下对姿态、空速、升降速度施加随机扰动

### 头部追踪控制
- **摄像头接入** — WebRTC getUserMedia 获取面部视频
- **像素级面部检测** — 基于亮度质心分析的面部定位算法，无需 FaceDetector API
- **姿态映射** — 头部左右移动控制横滚（±35°），上下移动控制俯仰（±25°）
- **自动校准** — 开启追踪后自动采集中性位置，空格键可随时重新校准
- **实时预览** — 画中画摄像头画面 + 面部检测框指示

### 飞行训练任务
- **自由飞行** — 无约束飞行，W/S 或 ↑/↓ 键控制油门
- **保持高度** — 需在指定高度保持 15 秒，偏差 ±200ft 内，实时评分
- **标准转弯** — 以标准转弯率（约 2.5°/s）完成 90° 转弯
- **仪表进近** — 沿 3° 下滑道下降至 100ft，模拟仪表着陆
- **任务反馈** — 达标/未达标弹出提示，3 秒后自动重新开始
- **快捷键** — Q/E 键循环切换训练任务，R 键重新开始当前任务

### 注意力分析
- **座舱内外占比** — 头部追踪时实时统计注视座舱内仪表 vs 座舱外视野的时间比例
- **注意力环形图** — 可视化座舱内外注视比例
- **12 仪表热力图** — 每个仪表的注视时长占比和最后注视时间，热区/警告标识
- **追踪状态指示** — 实时显示头部方向和当前俯仰/横滚角度

### 飞行场景
- **白昼巡航** — 晴朗蓝天 + 白云 + 清晰地面地形 + 跑道
- **夜间飞行** — 深色星空背景 + 暗色地面 + 仪表背光降低
- **湍流环境** — 灰蓝天空 + 随机姿态/空速扰动
- **低能见度** — 灰白雾霾 + 地面模糊 + 仪表亮度降低

### 系统监控面板
- **检测框标注** — 12 个仪表实时识别框 + 置信度标签
- **多区域感知** — 6 个座舱区域（主仪表板、顶置面板、中央操控台、发动机仪表、导航显示器、告警面板）的同步感知频率
- **机器人部署** — 3 台视觉采集单元在线状态及通信延迟，实时色彩编码
- **环境鲁棒性** — 6 种环境条件（正常光照、低光照、振动、遮挡、强反光、烟雾/雾霾）下识别精度条形图
- **全链路延迟分析** — 5 阶段延迟分解（图像采集/预处理/仪表检测/读数识别/融合输出）+ 60 秒实时趋势图
- **关键指标卡片** — 视觉采集延迟、仪表识别精度、多区域并发数、环境鲁棒性评分

## 使用方式

直接打开 `index.html`，或用任意 HTTP 服务器托管：

```bash
python -m http.server 3000
# 访问 http://localhost:3000
```

### 完整操作指南

| 操作 | 说明 |
|---|---|
| 点击"头部追踪"按钮 | 开启/关闭摄像头面部追踪控制 |
| 头部左右移动 | 控制飞机横滚（左右侧身） |
| 头部上下移动 | 控制飞机俯仰（抬头/低头） |
| 空格键 | 重新校准头部追踪中性位置 |
| W / ↑ 键 | 增加油门 |
| S / ↓ 键 | 减小油门 |
| 按键 1-4 | 切换飞行场景（白昼/夜间/湍流/低能见度） |
| 点击场景标签页 | 切换飞行场景 |
| 点击训练任务按钮 | 切换训练模式 |
| Q / E 键 | 循环切换训练任务 |
| R 键 | 重新开始当前训练任务 |

## 技术栈

- 纯 HTML/CSS/JS，零外部依赖
- Canvas 2D 渲染（座舱场景 + 仪表绘制 + 注意力环形图 + 延迟趋势图）
- 像素级亮度质心面部检测算法（色彩空间分析，无需 AI 模型）
- WebRTC getUserMedia 摄像头接入
- 飞行动力学物理模型（C172 简化）
- CSS Grid 响应式布局（1920px → 移动端三级适配）
- CSS 自定义属性（Design Tokens）统一主题

- 5.0版本新功能
